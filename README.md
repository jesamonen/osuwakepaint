# Osuwake Paints & Screeding Solutions

A modern, responsive multi-page website for showcasing paint and screeding products, completed projects, company services, and industry blog content.

## Features

* Responsive design
* Hero image slider
* Product catalog with search and filtering
* Projects gallery
* Blog section
* Paint quantity calculator
* Distributor registration form
* Contact page
* WhatsApp integration
* Dark/Light mode
* SEO-friendly structure

## Technologies

* HTML5
* CSS3
* JavaScript
* JSON

## Deployment

Hosted on GitHub Pages and optimized for desktop and mobile devices.
